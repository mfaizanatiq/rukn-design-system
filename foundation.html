Accessibility

Focus & Accessibility

Keyboard-accessible focus rings using the --ring token. Focus is only visible on keyboard navigation (:focus-visible), never on mouse click.

Accessibility Checklist

:focus-visible — Focus rings appear only on keyboard navigation
prefers-reduced-motion — Animations disabled for motion-sensitive users
Semantic colors — WCAG 2.1 AA contrast ratios across light/dark themes
sr-only utility — Screen-reader-only content class available
ARIA attributes — All components ship with proper roles and labels
/* Focus ring pattern */
:focus-visible {
  outline: none;
  box-shadow: 0 0 0 3px hsl(var(--ring) / 0.3);
  border-color: hsl(var(--ring));
}

/* Reduced motion */
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
