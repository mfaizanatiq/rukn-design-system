# Security Policy

## About

Rukn Design System is a client-side CSS and JavaScript library with no server-side code, no user data handling, and no authentication flows. The attack surface is minimal.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 2.2.x   | Yes       |
| < 2.2   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

- **Email:** mfaizanatiq@gmail.com
- **GitHub Issues:** [Open an issue](https://github.com/mfaizanatiq/RuknDesignSystem/issues)

We will acknowledge your report within 48 hours and provide a fix as soon as possible.
